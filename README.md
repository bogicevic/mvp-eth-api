# mvp-eth-api

> Base url of deployed live API: ```https://mvp-eth-api.herokuapp.com```

## To run the app locally:
1. `npm install`
2. `npm start`

## To run the app in hot-reload mode:
1. `npm install`
2. `npm start:dev`

## API doc:
`GET /balance` - to retreive the balance for hardcoded ETH address
